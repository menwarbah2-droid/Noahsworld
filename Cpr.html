How to solve this in Chrome Mobile:

  1. Grant Site Permission: Tap the Lock / Adjust Icon to the left of your Chrome URL address bar. Go to Permissions / Site Settings, find Camera and select Allow.
  2. Check Android Settings: Go to Phone Settings ➔ Apps ➔ Chrome ➔ Permissions ➔ Camera ➔ select Allow while using the app.
  3. Connect via HTTPS: Make sure you are using a secure context. Camera is blocked on raw HTTP connections by modern mobile browsers.
